Important: Windows SmartScreen Warning

Windows will show a security warning because this app is not digitally signed. This is normal for open-source software and doesn't mean the software is unsafe.

To install:

  1. Click "More info" on the Windows Defender SmartScreen warning
  2. Click the "Run anyway" button that appears
  3. Follow the installation wizard

Why isn't it signed? Code signing certificates cost $100-300/year. As an open-source project, we keep the software free instead of passing that cost to users. You can verify the app is safe by checking our open source code.
